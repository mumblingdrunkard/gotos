# Golang Teaching OS - gotos

A Teaching OS written in Golang.

Built around emulating a RISC-V CPU.

## Roadmap

- [ ] RV32I base instructions (40)
- [ ] RV64I base instructions (15)
- [ ] RV32M extension (8)

- [ ] RV32A extension (11)
- [ ] RV64A extension (22)
- [ ] RV32F extension (26)

- [ ] Zicsr extension (6)
