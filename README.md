# Golang Teaching OS - gotos

A Teaching OS written in Golang.

Built around emulating a RISC-V CPU.

## Roadmap

### Emulation

- [ ] RV32I base instructions
- [ ] RV64I base instructions
- [ ] RV32M extension
- [ ] RV64M extension
- [ ] RV32A extension
- [ ] RV64A extension
- [ ] RV32F extension
- [ ] RV64F extension
- [ ] Zicsr extension
- [ ] Zifencei extension

- [ ] MMIO

### OS

- [ ] Synchronization primitives
- [ ] C standard library
- [ ] Scheduler framework
- [ ] MMU framework

### Applications

- [ ] Shell
