# Golang Teaching OS - gotos

A Teaching OS written in Golang.

Built around emulating a RISC-V CPU.

## Roadmap

### Emulation

#### riscv32g target

- [ ] RV32I base instructions (40)
- [ ] RV32M extension (8)
- [ ] RV32A extension (11)
- [ ] RV32F extension (26)
- [ ] RV32D extension (26)
- [ ] Zicsr extension (6)
- [ ] Zifencei extension (1)

#### Peripherals

- [ ] MMIO

### OS

- [ ] Syscalls
- [ ] Synchronization primitives
- [ ] C standard library
- [ ] Scheduler framework
- [ ] MMU framework

### Applications

- [ ] Shell
